Politica de Privacidade

Ultima atualizacao: Junho 2025

1. Dados Coletados

Coletamos os seguintes tipos de dados:

  • Dados de cadastro: Nome de usuario, endereco de email, senha (criptografada com bcrypt)
  • Dados do jogo: Progresso, empresas, investimentos, ativos virtuais, historico de transacoes
  • Dados de uso: Logs de acesso, tipo de dispositivo, sistema operacional
  • Dados de publicidade: Identificador de publicidade do dispositivo (Google Advertising ID) para exibicao de anuncios

2. Como Usamos Seus Dados

  • Prover e manter o funcionamento do jogo
  • Personalizar a experiencia do jogador
  • Processar compras e recompensas
  • Exibir anuncios relevantes (Google AdMob)
  • Analise de erros e melhorias do app (Sentry)
  • Comunicacoes sobre o jogo (notificacoes)
  • Manter rankings e competicoes

3. Compartilhamento de Dados

Compartilhamos dados com os seguintes servicos terceiros apenas quando necessario:

  • Google AdMob: Para exibicao de anuncios personalizados
  • Stripe: Para processamento seguro de pagamentos
  • Sentry: Para monitoramento e correcao de erros
  • Railway: Hospedagem do servidor backend
  • MongoDB Atlas: Armazenamento seguro de dados

Nao vendemos seus dados pessoais a terceiros.

4. Seguranca dos Dados

  • Todas as comunicacoes sao criptografadas via HTTPS
  • Senhas sao armazenadas com hash bcrypt (nunca em texto puro)
  • Autenticacao via tokens JWT com expiracao
  • Acesso ao banco de dados restrito e protegido

5. Seus Direitos

Voce tem o direito de:

  • Acessar seus dados pessoais a qualquer momento
  • Corrigir dados incorretos ou desatualizados
  • Solicitar a exclusao completa da sua conta e dados
  • Revogar consentimento de uso de dados
  • Exportar seus dados em formato legivel

Para exercer esses direitos, acesse as configuracoes do app ou envie email para suporte@bizrealms.com

6. Exclusao de Conta e Dados

Voce pode excluir sua conta diretamente no app (Perfil > Configuracoes > Excluir Conta) ou por email.

Ao excluir a conta, todos os dados sao removidos imediatamente dos nossos servidores.

Para instrucoes detalhadas, visite nossa pagina de exclusao de conta.

7. Retencao de Dados

Seus dados sao mantidos enquanto a conta estiver ativa. Apos a exclusao, todos os dados sao removidos imediatamente e de forma irreversivel.

8. Menores de Idade

O BizRealms e destinado a maiores de 13 anos. Nao coletamos intencionalmente dados de menores de 13 anos. Se identificarmos uma conta de menor, ela sera removida.

9. Alteracoes nesta Politica

Esta politica pode ser atualizada periodicamente. Notificaremos os usuarios sobre mudancas significativas atraves do app ou por email.

10. Contato

Para duvidas sobre privacidade e dados:

Email: suporte@bizrealms.com
