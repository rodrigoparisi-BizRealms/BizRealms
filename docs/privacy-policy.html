Politica de Privacidade

Ultima atualizacao: Junho 2025

1. Dados Coletados

  • Dados de cadastro: Nome de usuario, email, senha (criptografada com bcrypt)
  • Dados do jogo: Progresso, empresas, investimentos, ativos virtuais
  • Dados de uso: Logs de acesso, tipo de dispositivo, sistema operacional
  • Dados de publicidade: Google Advertising ID para exibicao de anuncios

2. Como Usamos Seus Dados

  • Prover e manter o funcionamento do jogo
  • Personalizar a experiencia do jogador
  • Processar compras e recompensas
  • Exibir anuncios relevantes (Google AdMob)
  • Analise de erros e melhorias do app (Sentry)
  • Manter rankings e competicoes

3. Compartilhamento de Dados

  • Google AdMob: Para exibicao de anuncios
  • Stripe: Para processamento seguro de pagamentos
  • Sentry: Para monitoramento de erros
  • MongoDB Atlas: Armazenamento seguro de dados

Nao vendemos seus dados pessoais a terceiros.

4. Seguranca dos Dados

  • Comunicacoes criptografadas via HTTPS
  • Senhas com hash bcrypt
  • Autenticacao via tokens JWT
  • Acesso ao banco de dados restrito e protegido

5. Seus Direitos

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar exclusao da conta e dados
  • Revogar consentimento de uso de dados

Email: suporte@bizrealms.com

6. Exclusao de Conta

Exclua sua conta no app (Perfil > Configuracoes > Excluir Conta) ou por email.

Todos os dados sao removidos imediatamente.

Detalhes: pagina de exclusao de conta.

7. Menores de Idade

BizRealms e destinado a maiores de 13 anos.

8. Contato

Email: suporte@bizrealms.com
